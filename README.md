# Sales Performance & Incentive Analytics

## Overview
This project analyzes sales performance across regions and product categories and designs an incentive compensation model for sales representatives.

## Tools
- Python (Pandas, NumPy, scikit-learn)
- SQL
- Power BI
- Excel

## Features
- Sales performance analysis
- Region-wise growth trends
- Incentive compensation model
- Sales forecasting using linear regression
- Interactive Power BI dashboard

- Clothing category contributed the largest share of sales
